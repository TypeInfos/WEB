什么是HTML？

它是网页最基础的框架？

首先HTML并不是一种编程语言(markup language)，只是用来描述网页的一种标记语言。

HTML指的是超文本标记语言 (Hyper Text Markup Language)

标记语言是一套标记标签 (markup tag)

HTML使用标记标签来描述网页.

它就是网页最基础的框架.

HTML在web里面就想是劳动力，先准备原材料，并将原材料摆放整齐，等着设计师来分配.

HTML不是一种编程语言，所以更不需要基础，因为这就是基础.

CSS

网页的设计师

你了解了HTML,接下来就是CSS,它会让你的框架变成各种布局.让你变成一个名副其实的设计师,看起来就比较高端.

CSS 指层叠样式表 (Cascading Style Sheets)，样式表允许以多种方式规定样式信息.样式可以规定在单个的HTML元素中，在HTML页的头元素中，或在一个外部的CSS文件中。甚至可以在同一个HTML文档内部引用多个外部样式表.

你可以给字体设置各种样式,可以用透明度让颜色多样化.

其中比较难理解的属性就是定位，其中包括float(浮动),position(定位)

要理解定位,先理解何为文档流,并且什么时候脱离文档流,什么时候在文档流占据空间.

CSS3

一种神奇的语言

CSS3是高级的CSS，其中添加了过渡元素和3D视角，这让很多网页变得可观性,CSS3的推出和流行，使传统WEB发生巨大改变,动画是CSS3的核心.对于Web应用开发者而言，CSS3可以说是众望所归，这也是技术变更的硬性需求。在实际Web应用中新标准的采纳程度正在以令人目眩的速度不断地变更着，众多浏览器厂商也在不断加快对CSS3新特性的支持。

CSS3的优势:

在执效率来讲CSS3大于JS,所以一切能用CSS3独立完成的动画最好不要用JS

发展速度快，不会落伍，引领技术前端.

CSS3可以让你的个性得到发挥。

还有一种基于CSS却高于它的元语言:Sass

jQuery是一个兼容多浏览器的Javascript库

它的核心理念是write less,do more,写的更少，做的更多。以简单的代码做更好看，更实用的效果。

将原来的JS和CSS3的功能结合在一起，特别是操纵文档对象，选择DOM元素变得更敏捷，其模块化的使用方式使开发者可以很轻松的开发出功能强大的静态或动态网页。

jQuery插件是CSS3和JS爱好者开发出来的工具，让复杂动画变的简单化，让用户体验更舒畅。
